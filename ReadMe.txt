Задание написано на языке Java. Исполняемым файлом является InsertSortApp.jar.
Для запуска программы из командной строки используется следующая строка:
Полный путь до директивы с исполняемым файлом (в моем случае это был путь D:\Java\FocusStartPDNv.2\out\artifacts\InsertSortApp_jar) далее следует >java -jar InsertSortApp.jar in.txt out.txt -i -d
После имени файла идут 4 аргумента, большее или меньшее количество выведет ошибку.
В качестве первого аргумента(args[0]) идет имя файла, с которого считывается информация. Этот файл должен лежать в одной директории с исполняемым файлом. В моем случае он называется in.txt, вы можете создать любой другой, главное чтобы его имя в папке и аргумент в консоли совпадали. Файл может содержать данные только в кодировке UTF-8.
Второй аргумент(args[1]) это имя файла, в который записывает конечный результат. Имя может быть любым, файл создается в директории, где находится исполняемый файл.
Третий аргумент(args[2]) это метка типа, который содержится в файле. Значения могут быть "-i" для целых чисел и "-s" для строковых.
И последний, четвертый аргумент (args[3]) это метка для сортировки. Может принимать значения "-a" для сортировки по возрастанию и "-d" для сортировки по убыванию. 
К письму прилагаю 2 архива, один из которых содержит только исполняемый файл и текстовый файл, второй же содержит полностью весь проект.
